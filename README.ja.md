**Godot 4 で作られた、東アジア幻想風の見下ろし型ピクセルアート・アクションRPGプロトタイプ。**

明るい草地と冷たい月明かりを切り替えながら、古い村を歩き、剣を抜き、霊的な敵や炎をまとった獣と戦う小さなインディーゲーム実験です。武侠、仙侠、東洋ファンタジー、クラシックなトップダウンRPGの雰囲気を目指しています。

![Moonlit Jianghu day combat](docs/showcase/images/02-day-fire-lion-encounter.png)

## About

Moonlit Jianghu は完成品ではなく、遊べる状態で保存されたゲームプロトタイプです。昼夜切り替え、近接戦闘、敵との遭遇、インベントリUI、手作りの小さな村マップ、AI支援のスプライト制作ワークフローが含まれています。

## Features

- `P` キーで昼と夜を切り替え。
- 東アジア風 / 東洋ファンタジーの村、草地、土の道、屋根、木、岩、魔法の粒子。
- 武侠 / 仙侠風の剣士、怪異、修行幻想のムード。
- 見下ろし型アクション戦闘：移動、ダッシュ、攻撃、防御、スタミナ、ノックバック。
- 骸骨、ゾンビ、高速の敵、炎の獅子タイプ。

## Controls

- Move: `WASD` / arrow keys
- Attack: `J` / left mouse button
- Guard: `K`
- Dash: `L`
- Skill: `Space`
- Select skills: `1`-`5`
- Inventory: `M`
- Day / night: `P`
- Reset: `R`
